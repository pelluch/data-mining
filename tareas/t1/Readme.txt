Instrucciones

Ejecutar (con python v2.7) el archivo main.py en python/src
No recibe comandos adicionales por línea de comando, pero sí es necesario que esté
el la carpeta data en la misma posición relativa que tiene ahora para efectos de lectura de datos.
Se entrega un menu con las distintas opciones. El usuario puede ejecutarlas en cualquien orden y a opción,
y aparecen los plots pedidos.

Requerimientos
-----------------------------------------------
-----------------------------------------------
Python 2.7 (puede que funcione con la v3, pero no lo he probado)
Librería numpy disponible en el path de python
